# NP2 

Web application showcasing reductions between NP complete problems.

## Setup

Clone the repository.

```
git clone https://github.com/phatt-23/np2
cd np2
```

Install packages.

```
npm install
```

Run locally.

```
npm run dev
```

Build the application. 

```
npm run build
```

This will produce static files into `build/` directory.
The build directory contains `index.html`.
Serve these files.

For example, serve on homel.

```
scp -r build/* tra0163@homel.vsb.cz:~/public_html
```
